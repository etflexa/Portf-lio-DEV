# 💻 Portfólio DEV - Eduardo Flexa

## 🚀 Sobre o projeto

Este é o meu portfólio como desenvolvedor Fullstack, criado com o objetivo de apresentar meus projetos, habilidades e experiências na área de tecnologia.

O site foi desenvolvido com foco em **design moderno, responsividade e boa experiência do usuário**, permitindo que recrutadores e outros desenvolvedores conheçam melhor o meu trabalho.

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* Responsividade (Mobile First)

---

## 📌 Funcionalidades

* Apresentação pessoal
* Seção de projetos
* Seção de experiência
* Área de contato com links para redes sociais
* Layout responsivo para diferentes dispositivos

---

## 🎯 Objetivo

Este projeto foi desenvolvido para:

* Consolidar meus conhecimentos em desenvolvimento web
* Criar uma presença profissional online
* Servir como vitrine dos meus projetos
